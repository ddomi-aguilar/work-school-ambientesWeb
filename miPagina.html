ALGEBRA RELACIONAL

El álgebra relacional se considera un lenguaje de consulta procedimental, que representa los pasos de un conjunto de operaciones, donde intervienen, una o más relaciones de entrada, dando como resultado una nueva relación.

OPERADORES DERIVADOS

Se definen otros que se pueden expresar en función de los básicos, pero que expresan operaciones importantes dado que se usan habitualmente.

OPERACIONES DERIVADOS
SIMBOLO NOMBRE USO
Join Combina tuplas de dos relaciones basandose en la condicion de igualdad entre columnas especificas
Otros tipos son: inner join, left join, right join y full outer join
÷ Division Se utiliza para encontrar tuplas en una relacion que esten relacionadas con todas las tuplas de la otra relacion
Es una operacion mas avanzada requiere un enfoque especifico
Agregacion Se utiliza para calcular valores resumen de una columna en una relacion
SUM, AVG, COUNT, MAX, MIN
Diferencia de conjuntos Algunos sistemas de bases de datos tambien admiten operaciones de conjunto
UNION ALL (union que no elimina duplicados)
INSERSECT ALL (interseccion que permite duplicados)
Ordenamiento Se utiliza para ordenar las tuplas de una realcion segun los valores de una o mas columnas
ORDER BY
Subconsultas Permiten realizar una consulta dentro de otra consulta se puede utilizar en WHERE o en FROM

Estas operaciones adicionales amplian la funcionalidad del algebra relacional y permiten realizar consultas segun las nescesidades de los usuarios
